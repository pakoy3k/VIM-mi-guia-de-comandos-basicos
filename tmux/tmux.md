# tMUX

[Ctr + a] + detach

- return to attach window

> tmux attach  

- Show all sessions open 

> tmux ls

- Re-attach to specified session

> tmux attach -d -t <session id>
  
- Kill windows
  
> tmux kill-window -t window-number
